# 🏗️ GTX-B 노선 제2공구 지하수위 AI 예측 시연 대시보드

> **실제 공사 현장 계측 데이터 기반의 AI 지하수위 예측 및 시각화 일회성 시연 대시보드**
> 본 프로젝트는 터널 및 지하철 굴착 공사 시, 굴착 깊이(굴착고)와 시간 변화에 따른 주변 지하수위 변동을 예측하고 안전 관리자가 직관적으로 모니터링할 수 있도록 돕는 웹 기반 시연 시스템입니다.

---

## 🎯 최종 목표 (Project Goal)
도심지 터널(GTX-B 노선 제2공구) 및 지하철 굴착 공사 시, 지하수위의 급격한 저하는 주변 지반 침하(싱크홀) 및 붕괴 사고로 이어질 수 있는 치명적인 위험 요인입니다.
본 프로젝트의 최종 목표는 **실제 계측 수위계($W-1$ ~ $W-7$) 데이터**를 바탕으로, 미래 5일 동안의 수위 변화를 다각적 AI 모델로 비교 예측하고, 굴착 깊이 정보와 연동하여 현장 실무자가 직관적인 위험 감지 및 신속한 의사결정을 내릴 수 있는 **초경량 웹 대시보드를 시연하고 시스템을 검증하는 것**입니다.

---

## 🗺️ 시스템 아키텍처 (System Architecture)
서버 없이 브라우저 단독(Client-Side Only)으로 구동되는 초경량 구조입니다.

```
                      │
                      ▼ (안티그래비티 AI가 데이터 변환 및 예측 시뮬레이션)
┌──────────────────────────────────────────────────────────┐
│              Vite + React (Front-end App)                │
│                                                          │
│  ① Data Layer (내장형 가상 데이터베이스)                 │
│     - 엑셀 진짜 과거 데이터 + AI 가상 예측 데이터(5일치)  │
│                                                          │
│  ② State Layer (사용자 인터랙션 제어)                     │
│     - 현재 선택된 지하수위계 ID ($W-1$ ~ $W-7$) 저장     │
│                                                          │
│  ③ UI & Chart Layer (대시보드 화면)                      │
│     - 공사 현장 정보 배너 (GTX-B 노선 제2공구)           │
│     - 수위계 전환 버튼 7개                               │
│     - Recharts 기반 다중 축(Multi-Axis) 라인 차트        │
└──────────────────────────────────────────────────────────┘
```

## 🧠 AI 예측 모델 디테일 (AI Prediction Models)
본 시연에서는 시계열 예측 분야에서 널리 활용되는 3가지 계열의 핵심 알고리즘을 활용하여 미래 5일치의 지하수위 변화를 예측합니다.

### 1. ARIMA (AutoRegressive Integrated Moving Average)
* **개요**: 전통적인 통계학 기반의 시계열 분석 모델입니다.
* **수식**:
  $$\hat{Y}_t = c + \phi_1 Y_{t-1} + \dots + \phi_p Y_{t-p} + \theta_1 e_{t-1} + \dots + \theta_q e_{t-q}$$
* **특징**:
  - 과거 수위 자체의 자기회귀 성질(AR)과 이동평균(MA)을 결합하여 정상성 시계열의 트렌드를 추적합니다.
  - 급격한 외생 변수(예: 당일 굴착 깊이 급증)가 없는 평시 상태에서의 선형적이고 안정적인 수위 변동 추세를 매우 정확하게 모니터링합니다.

### 2. LSTM (Long Short-Term Memory)
* **개요**: 순환신경망(RNN)의 기울기 소실 문제를 극복한 딥러닝 모델입니다.
* **구조**: Forget Gate, Input Gate, Output Gate와 Cell State로 구성됩니다.
* **특징**:
  - 과거 계측 기록을 순차적으로 학습하며 수위 변동의 단기 패턴과 장기 패턴을 모두 포착합니다.
  - 다변량 입력(과거 지하수위 + 일별 굴착고 깊이)을 처리할 수 있어, 굴착 공정 진행 상황에 따른 비선형적 지하수위 하강 패턴을 효과적으로 모사합니다.

### 3. Transformer (Self-Attention)
* **개요**: Self-Attention 메커니즘을 시계열 예측에 도입한 최첨단 딥러닝 아키텍처입니다.
* **수식**:
  $$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$
* **특징**:
  - 시간적 거리에 상관없이 시계열 데이터 상의 특정 시점(예: 굴착고 급변 구간 등)에 높은 가중치를 주어 학습(Attention)합니다.
  - 복잡한 굴착 공정 스케줄 속에서 발생하는 돌발 수위 저하 현상이나 장기적인 지하수 흐름 교란을 정교하게 예측할 수 있습니다.

## 🚀 로컬 실행 방법 (How to Run Locally)
본 프로젝트를 다른 PC나 로컬 환경에서 클론하여 테스트하는 방법입니다.

```bash
# 1. 의존성 설치
npm install

# 2. 로컬 개발 서버 실행
npm run dev
```
브라우저에서 `http://localhost:5173` 으로 접속하여 시연용 대시보드를 바로 확인할 수 있습니다.
